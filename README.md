# 📊 Student Attendance Management System

## 📌 Description
A Java-based application designed to manage and track student attendance efficiently, reducing manual effort and improving accuracy.

## 🚀 Features
- Add, update, and delete student records
- Mark daily attendance
- View attendance reports
- Input validation for accuracy

## 🛠 Tech Stack
- Java
- OOP Concepts
- (Optional) MySQL, JDBC

## 🎯 Outcome
Improved attendance tracking efficiency and reduced manual errors.

## 📂 How to Run
1. Open project in IDE (Eclipse/IntelliJ)
2. Run main class
3. Follow console instructions
